# Инструкция по развертыванию на GitHub Pages

## Важно: Имя репозитория

Если ваше имя репозитория отличается от `favorite-car`, нужно обновить base path в следующих файлах:

1. **vite.config.ts** - измените `/favorite-car/` на `/ваше-имя-репозитория/`
2. **src/router/index.ts** - измените `/favorite-car/` на `/ваше-имя-репозитория/`

## Шаги развертывания:

1. Убедитесь, что все изменения закоммичены и запушены в репозиторий

2. В настройках репозитория GitHub:
   - Перейдите в Settings → Pages
   - Source: выберите "GitHub Actions"

3. При следующем push в main/master ветку автоматически запустится деплой

4. Или запустите вручную:
   ```bash
   npm run build
   ```
   Затем загрузите содержимое папки `dist` в ветку `gh-pages`

## Альтернативный способ (через gh-pages):

```bash
npm install --save-dev gh-pages
```

Добавьте в package.json:
```json
"scripts": {
  "deploy": "npm run build && gh-pages -d dist"
}
```

Затем выполните:
```bash
npm run deploy
```

## Проверка:

После деплоя сайт будет доступен по адресу:
`https://ваш-username.github.io/favorite-car/`

**Важно:** Если репозиторий называется по-другому, замените `favorite-car` на имя вашего репозитория во всех конфигурационных файлах.
