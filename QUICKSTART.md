# Быстрый старт

## Установка

```bash
npm install
```

## Запуск в режиме разработки

```bash
npm run dev
```

Откройте браузер по адресу: http://localhost:3000

## Основные команды

- `npm run dev` - запуск dev-сервера
- `npm run build` - сборка для продакшена
- `npm run preview` - предпросмотр продакшен-сборки
- `npm run lint` - проверка кода линтером

## Структура проекта

- `src/components/` - переиспользуемые компоненты
- `src/views/` - страницы приложения
- `src/router/` - маршрутизация
- `src/store/` - управление состоянием (Pinia)
- `src/services/` - API сервисы
- `src/types/` - TypeScript типы

## Примечания

1. Для работы с API необходимо настроить backend или использовать mock-данные
2. По умолчанию API URL: `http://localhost:3001/api`
3. Избранное сохраняется в localStorage
4. Авторизация также использует localStorage для хранения токена

## Следующие шаги

1. Настройте backend API или используйте mock-данные для тестирования
2. Настройте переменные окружения в `.env` файле
3. Кастомизируйте стили под ваш бренд
4. Добавьте дополнительные функции по необходимости
